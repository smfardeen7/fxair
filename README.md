## Logo

Place your FXAIR logo as `public/logo.png`. If the file is missing, the header shows a text fallback (FX + AIR in brand colors).

## Build

```bash
npm run build
npm run preview
```

## Stack

- **Vite** + **React** + **TypeScript**
- **Tailwind CSS** (FXAIR colors: orange, blue, yellow, gray)
- **DM Sans** + **Syne** (Google Fonts)

Sections: Hero, Trust bar, About, Stats, Features, How it works, Benefits, Testimonials, CTA, Pricing, FAQ, Footer.
